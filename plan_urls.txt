# Direct download links to Vinmonopolet launch-plan Excel files — one URL per line.
# This is the reliable way to add historical plans (the site is a SPA, so auto-scraping
# a page's HTML may miss the link).
#
# HOW: on vinmonopolet.no, open the "Lanseringer" section and its launch-plan archive,
# right-click each Excel download link -> "Copy link", and paste the URL here. Prefer the
# ENGLISH edition. (Exact archive paths aren't listed here on purpose — grab the real
# links yourself so nothing is guessed.)
#
# Then locally:  python3 fetch_launch_plans.py && python3 build_app.py
# Or just commit this file — the refresh-launch-plans workflow does the rest monthly.
#
# Lines starting with # are ignored. Paste real .xlsx URLs below:
#
# Vinmonopolet publishes these on its Sanity CDN with opaque hash filenames (no year in
# the URL), so fetch_launch_plans.py derives each plan's period (YYYY-half) from the launch
# months inside the file, and skips any period already present as a specs_*.json. Only the
# .xlsx tender tables go here; the companion PDFs (submission info) aren't parseable plans.
